# Futuristic Social Links Page

A modern, responsive social links page with a futuristic design featuring neon glow effects, glassmorphism, and smooth animations.

## Features

- Fully responsive design for mobile and desktop with optimized touch support
- Dark theme with neon glow effects
- Glassmorphism UI elements
- Animated hover effects and transitions
- Circular profile avatar (loaded from Images/Logo.svg)
- Semi-transparent design elements with blur effect
- Animated GIF background (Images/background.gif)
- Semi-transparent buttons that become slightly gray on hover and scale up
- Social media buttons with icons for LinkedIn, Instagram, Telegram, Telegram Channel, VK, and GitHub
- Modern CSS using flexbox for layout
- Single HTML file with embedded CSS and JavaScript
- Font Awesome icons for social networks

## Technologies Used

- HTML5
- CSS3 (with flexbox, gradients, and animations)
- JavaScript (for interactive effects)
- Font Awesome (for icons)

## Design Elements

- Animated neon borders
- Glass-like transparency effects
- Subtle hover animations
- Scaling effects on interaction
- Gradient backgrounds
- Blur effects for depth
